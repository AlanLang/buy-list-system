# 文讯科技B/S基础框架
## 功能描述

* 登陆
* 数据库访问
* 菜单角色
* 页面内权限
* 弹窗

## 依赖包

| 名称      | 描述                                                         |
| --------- | ------------------------------------------------------------ |
| LayUI  | 前端框架   |
| MiniProfiler.EF6   | 数据库访问工具 |
| Newtonsoft.Json  | JSON序列化  |
| Microsoft.AspNet.Mvc   | MVC框架  |
